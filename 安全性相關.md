## 安全性相關

### 1. 如何避免 AI 執行危險指令

- VS Code 黑/白名單設定 `chat.tools.terminal.autoApprove` 
  (黑名單僅是要求使用者允許，並非禁止執行)

- 可以用 hook 阻擋特定的指令
- 重要檔案做好版本控制 git

- **/autopilot /goal 會允許所有指令**
- `chat.assistedPermissions.enabled` ，agenthost 才有

### 2. 外部資源使用

- 接第三方模型時 要留意資料是否會被拿去訓練使用，ex.MYAI168、Nvidia
- 網路上的 skill、MCP等資源 要確認過再使用，避免可能的 Prompt Injection
